Para inserir equações, é preciso usar uma notação específica (MathJax) e colocá-la entre dois \$$.

Por exemplo, o código

```markdown
$$P(m,n)=\sum_{j=1}^J \frac{N_{jm}}{N_m}\{ \frac{L_{jn}}{L_j} \}$$
```

Resultaria em:

$$P(m,n)=\sum_{j=1}^J \frac{N_{jm}}{N_m}\{ \frac{L_{jn}}{L_j} \}$$
